## 写作原则

- 每篇博客应有**独立的主题角度**，与已有文章形成差异化
- 面向目标读者，技术深度与读者背景匹配
- 优先选择有教育价值的主题——即使读者不用 catpaw，读完也能学到东西
- catpaw 的推广应自然融入内容，而非硬广
